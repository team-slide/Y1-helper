Windows Driver Installation

Windows x64 (64-bit)

Step 1: Install MediaTek SP Flash Tool Driver

This driver is required for firmware installation on MediaTek devices.

Download MediaTek Driver

After downloading, extract the ZIP file and run the installer as Administrator.

Step 2: Install UsbDk Driver

This driver provides enhanced USB communication capabilities.

Download UsbDk Driver

Run the MSI installer as Administrator and follow the installation wizard.

Step 3: Restart Your Computer

After installing both drivers, restart your computer to ensure they're properly loaded.

With both drivers installed, you'll have access to all firmware installation methods including guided installation, MTKClient, and SP Flash Tool.

If you haven't already, installing Homebrew can help with Python dependencies:

Python Requirements

Make sure you have Python 3.7+ installed. You can check by running:

python3 --version

Linux Setup

üéâ Great News!

Linux doesn't require any special drivers for Innioasis Updater. The app works natively on Linux with all features available.

Install Required Packages

You may need to install some additional packages depending on your distribution:

Ubuntu/Debian:

sudo apt-get install python3-pip python3-venv libusb-1.0-0-dev

Fedora/RHEL:

sudo dnf install python3-pip python3-venv libusb1-devel

Arch Linux:

sudo pacman -S python-pip python-virtualenv libusb

USB Permissions

You may need to add your user to the dialout group for USB access:

sudo usermod -a -G dialout $USER

Log out and back in for the changes to take effect.
