🪟 Windows Installation

Getting Started

Before we can install the updater, you'll need two drivers so your computer can talk to your Y1 properly:

MediaTek Driver: This lets your computer recognize your Y1 device

USB Development Kit: This helps with USB communication

Install & Run

Now let's get the actual updater app:

Almost Done!

Here's what to do next:

  1. Install the MediaTek driver you downloaded
  2. Install the USB Development Kit
  3. Install the Innioasis Updater
  4. Restart your computer
  5. Launch Innioasis Updater from the Start Menu

Important: Don't skip the restart - it makes sure everything works properly!

🍎 macOS Installation

First Things First

Before we can install the updater, you'll need Homebrew on your Mac.

Think of Homebrew as a helper that makes sure all the pieces work together properly.

Grab the App

Now let's download the actual updater app. Click the button below and it'll start downloading to your Downloads folder:

Almost Done!

Now let's get the app installed:

  1. Double-click the downloaded file (it'll look like a disk image)
  2. Drag the Innioasis Updater app to your Applications folder
  3. Open it from your Applications folder

Important: You might need to tell your Mac it's okay to run this app in System Preferences.
