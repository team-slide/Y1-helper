# Y1 Helper is now called Y1 Remote Control and comes with Innioasis Updater for Windows. Available at www.innioasis.app
# [Download at innioasis.app](https://innioasis.app)
<img src="https://github.com/team-slide/Innioasis-Updater/blob/main/mtkclient/gui/images/icon.png" alt="Innioasis Updater Icon" width="128"/>
Innioasis Updater is an easy, one-click firmware installer for the Innioasis Y1 MP3 player (may support more models in the future. It helps you install Updates, Factory Restore and install Custom Firmwares with Rockbox. 
Windows users can still access the existing Y1 Helper remote control and .apk installation features if they need to.
