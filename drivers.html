📥 MediaTek Driver Installation Info

When installing the MediaTek driver, you may see a dialog asking about removing the "origin driver". Here's what to do:

Important: If the installer asks "Remove origin driver?", click Yes. This is normal and helps ensure the driver works properly with your Y1 device.

This step is necessary because:

  • It removes any conflicting drivers that might interfere
  • It ensures the MediaTek driver has full control over USB communication
  • It prevents driver conflicts that could cause connection issues

After installation, restart your computer as prompted to complete the setup.

🪟 Windows ARM64 - Alternative Setup

⚠️ Limited Functionality on ARM64

Windows ARM64 devices can't run the full firmware installation features due to driver limitations. However, you can still use Remote Control, Screenshot, and APK installation features.

Option 1: Use Available Features

You can still download and use Innioasis Updater for the features that work:

  • 📱 Remote Control - Control your Y1 from your computer
  • 📸 Screenshot - Take screenshots of your Y1
  • 📦 APK Installation - Install Android apps on your Y1

Option 2: Advanced Users - WSLg Setup

If you're comfortable with advanced setup, you can try running the Linux version in WSLg (Windows Subsystem for Linux with GUI):

📥 USBIPD-Win Driver

First, install this driver to enable USB passthrough to WSLg:

🔧 Show WSLg Setup Guide (Advanced)

🐧 WSLg Setup for Windows ARM64

Step 1: Install WSLg

First, make sure you have WSLg enabled on your Windows ARM64 device:

  1. Open PowerShell as Administrator
  2. Run: wsl --install
  3. Restart your computer
  4. Install Ubuntu from the Microsoft Store

Step 2: Install USBIPD-Win

You should have already downloaded this. Install the .msi file you downloaded earlier.

Step 3: Run Setup Script

In your Ubuntu WSLg terminal, run this command to automatically set up everything:

curl -fsSL https://raw.githubusercontent.com/team-slide/Innioasis-Updater/main/.web_scripts/setup_wslg_arm64.sh | bash

This script will:

  • Install required packages
  • Configure USBIPD for your Y1
  • Download and set up Innioasis Updater
  • Guide you through the first connection

Step 4: Connect Your Y1

After the script completes:

  1. Connect your Y1 via USB
  2. In Windows, run: usbipd wsl list
  3. Attach your Y1: usbipd wsl attach --busid [YOUR_BUS_ID]
  4. In WSLg, run: innioasis-updater
